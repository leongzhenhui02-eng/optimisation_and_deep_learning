# optimisation_and_deep_learning
Sentiment Analysis
1.	Performed extensive data preprocessing on ~50,000 text records, including text cleaning, tokenization, stop word removal and normalization
2.	Conducted exploratory data analysis (EDA) to examine sentiment distribution, text length patterns and word frequency trends
3.	Developed data visualizations (word clouds, histograms, bar charts) to uncover key linguistic patterns in positive vs negative reviews
4.	Prepared data for modelling through label encoding, sequence padding and train-test splitting 
5.	Built and evaluated deep learning models for sentiment classification
6.	Applied Random Search optimization to improve model performance and reduce overfitting
7.	Evaluated models using accuracy, precision, recall, F1-score and confusion matrix to ensure robust performance
